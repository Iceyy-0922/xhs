# xhs
A chat feature on the web version of Xiaohongshu.
This Demo is a prototype of the real-time conversation function designed for the Weishixin web version. It focuses on addressing the pain point of being unable to directly communicate via private messages on the web, and has connected the links between content browsing and interactive consultation.

Key features: 1. A lightweight entry integrated with notifications; 2. A dual-column conversation layout suitable for large-screen scenarios; 3. Exclusive experiences such as quick templates, streaming messages, and note card sending, which cater to both users' consultation needs and creators' efficient operation requirements.
